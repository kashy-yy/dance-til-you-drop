# dance-til-you-drop
A rhythm-based boss fight game built in Godot using C#.

## About

Dance Boss is a game where the player defeats a boss by correctly performing dance move patterns on a tiled dance floor. Each successful dance deals damage, while stepping on dangerous tiles or making mistakes can harm the player.

This project is being developed as a first-year Computer Science project to practise object-oriented programming, data structures, and game development with Godot.

## Features

- 9 × 10 procedurally generated dance floor
- Tile-based player movement
- Randomised dance floor colours
- Tile state system (Safe, Warning, Danger, Off)
- Built using Godot and C#

## Planned Features

- Boss battle
- Dance move pattern system
- Pattern display above the dance floor
- Player move history checking
- Dangerous floor mechanics
- Health system
- Music and visual effects
